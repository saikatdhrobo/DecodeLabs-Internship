# Data Classification Using AI

## Project Overview

This project demonstrates a basic AI classification model using the Iris Dataset and the K-Nearest Neighbors (KNN) algorithm.

## Dataset

- Iris Dataset
- Total Samples: 150
- Classes:
  - Setosa
  - Versicolor
  - Virginica

## Technologies Used

- Python
- Scikit-Learn
- NumPy
- Pandas

## Features

- Load Iris Dataset
- Data Scaling using StandardScaler
- Train/Test Split (80/20)
- KNN Classification (K=5)
- Model Training
- Prediction
- Performance Evaluation

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- Classification Report

## Output

Accuracy: 100%

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python main.py
```
