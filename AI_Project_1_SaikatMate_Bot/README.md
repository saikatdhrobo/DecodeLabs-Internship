# SaikatMate Bot

## Project Overview

SaikatMate Bot is a simple rule-based AI chatbot developed using Python. It responds to predefined user inputs using a dictionary-based knowledge base. The chatbot runs continuously until the user enters an exit command.

## Features

- Rule-based chatbot
- Continuous conversation using a while loop
- Input sanitization (lower() and strip())
- Dictionary knowledge base
- 15 predefined intents
- Friendly fallback response
- Personalized conversation using the user's name
- Multiple exit commands (exit, stop, bye, quit)

## Technologies Used

- Python 3

## How to Run

1. Open the project folder.
2. Run the following command:

```bash
python main.py
```

3. Enter your name.
4. Start chatting with SaikatMate Bot.
