# Project Documentation

## Project Name

SaikatMate Bot

## Objective

The objective of this project is to build a simple rule-based chatbot using Python dictionaries instead of multiple if-else statements.

## Requirements Implemented

- Continuous while loop
- User name input
- Input sanitization
- Dictionary knowledge base
- Dictionary .get() method
- Friendly fallback response
- Exit commands
- Personalized responses

## Working Process

1. Ask the user for their name.
2. Welcome the user.
3. Take user input.
4. Convert input to lowercase.
5. Remove extra spaces.
6. Check exit commands.
7. Search the dictionary.
8. Display the response.
9. Repeat until the user exits.

## Tools Used

- Python
- VS Code

## Conclusion

This project demonstrates the basic concepts of Python programming, including loops, dictionaries, user input, string manipulation, and simple chatbot logic.
